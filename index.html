CSCE 242: Web Applications

Aditya Padala

Class description: Learn how to develop web pages.

Special note: With a kind and energetic professor.

Assignments

  • Assignment 1: Basic HTML

    Assignment 1 involves creating a main webpage as a portal for class assignments with a specific directory structure, and an Assignment 1 page on a chosen topic that includes various HTML elements like headings, paragraphs, images, a list, a table, and external links, along with the required structural HTML code.

  • Assignment 2: Basic CSS

    Assignment 2 involves enhancing the main portal webpage with styled content and creating a new, themed webpage with various HTML elements like navigation links, paragraphs, images, and a table, all cohesively styled using different CSS techniques (element, ID, class name) to ensure a visually appealing and consistent design.

  • Assignment 3: Page Layout

    Assignment 3 involves creating a single webpage using HTML5 elements (header, footer, nav, section, aside, article) structured traditionally with a header/content/footer layout, and styled with CSS Flexbox. The design includes a responsive navigation bar, column layout adjustments via media queries for different screen sizes, and visual enhancements like background colors and padding, following a provided diagram.

  • Assignment 4: Recreate CSS Page

    Assignment 4 involves recreating a specific University of South Carolina webpage. The task requires employing HTML5 elements (header, footer, nav, section, aside, article) in a traditional header/content/footer structure, coupled with CSS Flexbox for styling. The webpage must feature a responsive navigation bar that remains visible on small screens, use placeholder images, and have all text replaced with Lorem Ipsum. Additionally, all hyperlinks should redirect to '#', and the color scheme must match the original page using a color picker tool. The assignment is designed to enhance the ability to accurately replicate HTML/CSS layouts, with particular emphasis on flexbox utilization for responsive design.

  • Assignment 5: JavaScript, Buttons, Functions, and more

    Assignment 5 involves designing an interactive web page that incorporates three main features: a click-triggered image change with size and aspect ratio consistency, a slider-driven full-circle image rotation for a dynamic visual effect, and an interactive column where users can add star images with each click, ensuring a neat and visually appealing arrangement. The goal is to blend seamless functionality with an intuitive and engaging user interface.

Projects

  • Semester Project Part 1: Topic Selection

    Project part 1 is a PDF document that serves as a blueprint for a custom website, encapsulating the creator's skills and creativity. It outlines the website's name, provides an insightful description of the chosen topic, and lists the main pages, reflecting the site's navigational structure. Furthermore, it details the data management approach, including data types, security measures, and administrative functionalities. This project not only highlights the developer's technical prowess but also offers the flexibility to craft a personalized online webiste.

  • Semester Project Part 2: Wireframes

    Project part 2 involves creating detailed, professional wireframes for every page of the website, ensuring consistency, alignment, and uniformity in design elements. It includes crafting specific detail pages for certain items and using a monochrome color palette, with placeholder content where necessary.
