CSCE 242: Web Applications

Aditya Padala

Class description: Learn how to develop web pages.

Special note: With a kind and energetic professor.

Assignments

  • Assignment 1: Basic HTML

    Assignment 1 involves creating a main webpage as a portal for class assignments with a specific directory structure, and an Assignment 1 page on a chosen topic that includes various HTML elements like headings, paragraphs, images, a list, a table, and external links, along with the required structural HTML code.

  • Assignment 2: Basic CSS

    Assignment 2 involves enhancing the main portal webpage with styled content and creating a new, themed webpage with various HTML elements like navigation links, paragraphs, images, and a table, all cohesively styled using different CSS techniques (element, ID, class name) to ensure a visually appealing and consistent design.

  • Assignment 3: Page Layout

    Assignment 3 involves creating a single webpage using HTML5 elements (header, footer, nav, section, aside, article) structured traditionally with a header/content/footer layout, and styled with CSS Flexbox. The design includes a responsive navigation bar, column layout adjustments via media queries for different screen sizes, and visual enhancements like background colors and padding, following a provided diagram.
